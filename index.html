Introduction

Welcome to the IBS KYC API! You can use our API to access IBS Simplified Forms API endpoints, which can get information on user account details and forms queries in our database.

We have language bindings in PHP, Shell and JavaScript! You can view code examples in the dark area to the right, and you can switch the programming language of the examples with the tabs in the top right.

Authentication

To authorize, use this code:

$curl = curl_init("api_endpoint_here");

curl_setopt($curl, CURLOPT_USERPWD, "username:password");

# With shell, you can just pass the correct header with each request
curl "api_endpoint_here"
  -H "Authorization: Basic username:password"

const authToken = require('api_endpoint_here');

let api = authToken.authorize('username:password');

Make sure to replace authToken with your API key.

OAuth uses API keys to allow access to the API. You can register a new authToken API key at our developer portal.

OAuth expects for the API key to be included in all API requests to the server in a header that looks like the following:

Authorization: bearer 'authToken'

You must replace authToken with your personal API key.

Accounts

Get All Users

require 'users'

api = Users::APIClient.authorize!('users')
api.users.get

curl "https://ibsettle.com/kyc/api/users/get/all"
  -H "Authorization: bearer 'authToken'"

const users = require('users');

let api = users.authorize('users');
let users = api.users.get();

The above command returns JSON structured like this:

{
  "id": 1,
  "token": "0291f97f835756e953b98eccbb55dca9",
  "email": "example@example.com",
  "created": "2019-02-14 16:00:00",
  "modified": "2019-02-14 17:00:00"
},
{
  "id": 2,
  "token": "a541227f835746e953b22efctb25dce2",
  "email": "example@example.com",
  "created": "2019-02-14 16:00:00",
  "modified": "2019-02-14 17:00:00"
}

This endpoint retrieves all users.

HTTP Request

GET https://ibsettle.com/kyc/api/users/get/all

Remember — a happy user is an authenticated user!

Get User by token

require 'users'

api = Users::APIClient.authorize!('users')
api.users.get("a541227f835746e953b22efctb25dce2")

curl "https://ibsettle.com/kyc/api/users/get/byToken/a541227f835746e953b22efctb25dce2"
  -H "Authorization: bearer 'authToken'"

const users = require('users');

let api = users.authorize('users');
let max = api.users.get("a541227f835746e953b22efctb25dce2");

The above command returns JSON structured like this:

{
  "id": 2,
  "token": "a541227f835746e953b22efctb25dce2",
  "email": "example@example.com",
  "created": "2019-02-14 16:00:00",
  "modified": "2019-02-14 17:00:00"
}

This endpoint retrieves a specific user.
